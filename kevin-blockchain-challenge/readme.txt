Require: Nodejs
For download Nodejs:
1. Go to https://nodejs.org/en/download/, download the file that suit your operation system and install.
2. Download any code editor on your preference, open terminal in code editor and type in “npm install” and run.
3. To create a folder with React environment, type in “npx create-react-app <your_file_name>” and run. Eg: npx create-react-app blockchain-challenge.
4. To run the app, type in “npm start”.

You now have the folder with React environments and ready to develop your React App, you can add or edit the files inside the folder, to see the result, open browser and type in “localhost:3000”.
